# Class Management System

## Description
This project is a **Gym Management System** that allows trainers to add classes, register members, and view class registrations, as well as admins monitoring the trainers' work through dashboards. The system is built using Java Swing for the graphical user interface and utilizes JCalendar 0.8.1 for date selection when required.

## Features
- **Add Class:** Create new classes with details like Class ID, Name, Trainer ID, Duration, and Maximum Participants.
- **Register Member:** Assign members to classes with a registration date.
- **View Registrations:** Display a list of all registered members along with their assigned classes and registration dates.

## Technologies Used
- Java Swing (GUI)
- JCalendar 0.8.1 (for date selection)

## Installation and Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/Mazen-Azhary/GymManagementSystem
   ```
2. Open the project in your Java IDE (e.g., IntelliJ, Eclipse, or NetBeans).
3. Ensure **JCalendar 0.8.1** is added to the project dependencies.
4. Compile and run the `Frontend/MainWindow.java` file to launch the system.
